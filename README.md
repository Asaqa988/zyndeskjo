# Zyndesk — AI Solutions & Automation Studio

Dynamic, bilingual (EN/AR with full RTL) marketing website built with **React + Vite + Framer Motion**, designed with the **UI/UX Pro Max** skill (AI-Native UI style, dark glassmorphism) and inspired by 21st.dev.

## Quick start

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
```

A ready production build is already included in `dist/` — open `dist/index.html` or deploy the folder to Netlify/Vercel/any static host.

## What's inside

- **Hero** — kinetic gradient headline, animated aurora orbs, parallax on scroll, live stats
- **Tech marquee** — infinite Framer Motion loop of the stack (OpenAI, Claude, LangChain, Odoo, Flutter…)
- **Services** — bento grid, 8 service lines, scroll-reveal + hover lift
- **Demo showcase** — AI-generated product images of the Zyndesk helpdesk dashboard & mobile app
- **Live AI chat demo** — interactive scripted agent ("Zyn") with typing indicator, works in both languages
- **Process, Contact, Footer** — Amman · University Street, +962 7 9770 0235, [LinkedIn](https://www.linkedin.com/company/careerak1/)
- **EN ⇄ عربي toggle** in the navbar switches language and flips the layout to RTL

## Design system

See `DESIGN-SYSTEM.md` — generated by the UI/UX Pro Max reasoning engine (style: AI-Native UI, typography: Space Grotesk, effects: typing indicators, pulse animations, smooth reveals). The pre-delivery checklist is implemented: SVG icons only, cursor-pointer, 150–300ms hover transitions, visible focus states, `prefers-reduced-motion` respected, responsive at 375/768/1024/1440px.
